Backup y restauración

Descargá una copia completa de todos tus datos (clientes, productos, mano de obra, facturas, presupuestos, stock, agenda y configuración) junto con tus imágenes (logo, fondo y fotos de productos). Guardá el archivo en un lugar seguro fuera del NAS.

Restaurar un backup reemplaza todos los datos actuales por los del archivo. Esta acción no se puede deshacer.

Backup automático a Google Drive

Conectá una cuenta de servicio de Google para que la app suba un backup completo a tu Drive cada cierta cantidad de horas, aunque no tengas la app abierta. Pasos:

  1. Entrá a Google Cloud Console, creá un proyecto (o usá uno existente) y creá una cuenta de servicio.
  2. Generale una clave nueva en formato JSON y descargala.
  3. Habilitá la Google Drive API para ese proyecto.
  4. En tu Google Drive, creá (o elegí) una carpeta para los backups, compartila con el email de la cuenta de servicio (algo como nombre@proyecto.iam.gserviceaccount.com) dándole permiso de Editor, y copiá el ID de la carpeta (lo que aparece en la URL después de /folders/).
  5. Subí el JSON acá abajo, pegá el ID de carpeta y activá el backup automático.
